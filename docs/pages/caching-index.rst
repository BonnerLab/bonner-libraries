Installation
------------

To use the development version of the package, run:

``pip install git+https://github.com/BonnerLab/bonner-caching``

Since the package is under active development, we have not yet released a stable version.

Environment variables
---------------------

All cached outputs are stored within the directory specified by ``$BONNER_CACHING_CACHE``.
